I-SEE: An Android Application for Visually Impaired Users

Conceptualized and developed an Android app for visually impaired users using open-source APIs.

PROJECT HIGHLIGHTS

Role

Android Developer

Duration

Jul 2018 - Oct 2018

Technologies

Java, Firebase Vision Text Recognizer, TextToSpeech API

Results

Improved accessibility for visually impaired users with integrated features such as reading text, magnification, speed dial, and SOS functionality.

ABOUT THE PROJECT

Background: I conceptualized and developed the I-SEE Android App for visually impaired users, focusing on enhancing daily activities and accessibility. The app uses open-source APIs like Firebase Vision Text Recognizer and TextToSpeech API to assist users in reading text, magnifying content, making quick calls through speed dial, and sending emergency SOS messages.

Timeline: Jul 2018 - Oct 2018

My Role: As the Android Developer, I was responsible for the overall design and development of the app, integrating features that allow visually impaired users to navigate their environment with greater ease.

Tools used: Java, Firebase Vision Text Recognizer, TextToSpeech API
