INFINITI Website

Revamped the global INFINITI website to enhance its digital presence with a modern and interactive design, improving the user experience for a global audience.

PROJECT HIGHLIGHTS

Role

Frontend Engineer

Duration

6 months

Technologies

React (TypeScript), AEM CMS, HTML, CSS, SEO, Webpack, Git

Results

25% increase in site visits and 10% increase in online orders

ABOUT THE PROJECT

Background: I worked on the Nissan INFINITI website revamp as part of a global effort to redefine and enhance the brand’s digital presence. INFINITI aimed to position itself as a premium automotive brand, requiring a website with a modern, seamless, and interactive user experience. The challenge was to implement sophisticated animations, optimize performance, and meet high aesthetic and technical expectations.

Timeline: 6 months

My Role: As a Software Engineer, I contributed to the frontend development, ensuring smooth integration of complex animations and UI components using React and TypeScript. I optimized page load times and performance, leading to a 25% increase in site visits and a 10% increase in online orders.

Tools used: React, TypeScript, Redux, GSAP, Webpack, AWS
