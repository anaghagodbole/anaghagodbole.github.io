StockWise - Smart Stock Trading & Investment Platform

StockWise is a comprehensive web application designed for stock trading, portfolio management, and market analysis

PROJECT HIGHLIGHTS

Role

Full Stack Engineer

Duration

4 months (Jan 2025 – Apr 2025)

Technologies

React, Vite, Redux Toolkit, Node.js, Express.js, MongoDB, Redis, Stripe API, OpenRouter API, Bootstrap

Results

Deployed a feature-rich trading platform with AI integration, enabling 100+ simulated transactions and user engagement through real-time insights.

ABOUT THE PROJECT

Background: I built StockWise, a full-stack web application for stock trading, portfolio management, and market analysis. It allows users to track real-time market data, manage investments, and interact with an AI-powered chatbot for financial insights. The platform serves three roles: users, brokers, and admins—each with customized access and features.

Timeline: 4 months (Jan 2025 – Apr 2025)

My Role: As the sole developer, I implemented both frontend and backend functionalities. I built features including user profile management, wallet top-ups via Stripe, portfolio and transaction handling, buy/sell stock operations, watchlist management, and a personalized dashboard. I also integrated an AI-powered chatbot (via OpenRouter) for stock-related queries, and ensured a responsive UI across all user roles.

Tools used:
Frontend: React, Vite, Redux Toolkit, React-Bootstrap, Formik, Recharts, Axios, Stripe.js, Google OAuth
Backend: Node.js, Express.js, MongoDB, Mongoose, JWT, Multer, Redis, Nodemailer, Stripe API
Styling: Bootstrap, Custom CSS
APIs: Yahoo Finance/Finnhub (Market Data), OpenRouter (AI Chatbot)
Misc: ESLint, dotenv, CORS
